# Linqpal Task
A interview assignment from linqpal
## Clone 

In terminal:   
```js   
$ git clone <repository clone url>
$ cd linqpal-task
```
## Client Setup   
In new terminal:   
```js 
$ cd client

$ npm i

$ npm run build

$ npm run serve

// Client is running at 3000 port
// Open http://localhost:3000 in browser
```
